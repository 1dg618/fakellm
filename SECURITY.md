# Security Policy

## Supported Versions

FakeLLM is in early development. Only the latest version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in FakeLLM, please **do not** open a public issue.

Instead, report it privately by:

1. Going to the [Security tab](https://github.com/1dg618/fakellm/security) of this repository
2. Clicking "Report a vulnerability"

Alternatively, you can email the maintainer directly (add your email here if you want).

You can expect:
- An acknowledgment within a few days
- An update on the status within a week
- A fix or mitigation as soon as reasonably possible for confirmed issues

Thanks for helping keep FakeLLM and its users safe!
